# Travel Website (Node.js)

A travel website built with Node.js following an MVC structure.  
The project allows users to explore travel destinations and view travel-related information through a web interface.

This project was developed as a learning and practice project for backend development using Node.js.

---

## Features
- Display travel destinations
- View travel details
- MVC architecture (Model – View – Controller)
- Static assets handling (CSS, images)
- Database seeding for initial data

---

## Tech Stack
- **Backend:** Node.js, Express.js  
- **View Engine:** EJS  
- **Database:** MongoDB  
- **Frontend:** HTML, CSS, JavaScript  
- **Others:** dotenv, nodemon
